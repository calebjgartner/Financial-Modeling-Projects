Brief description for the following models:

Financial Forecast Model:
  - This financial forecast is pretty cut and dry, the assumptions are fair given market conditions at the time of the forecast. Shows simple debt schedule and discounted cash flows model also.
Hedged vs Unhedged Foreign Currency Model
  - This excel model demonstrates the hedged vs unhedged returns of a foreign investment based on varying conditions of the investment returns and inflation.
  - Completed for a graduate project during my international business class for Fall 2025.
LBO Model
  - This excel model is a variant of the base financial forecast from earlier.
  - Demonstrates the new capital structure of a leveraged buyout from a private equity sale.
  - Includes a "LBO Returns" tab that shows the total return on investment for investors.
Markowitz Optimizer Model
  - Macro-enabled as this is a portfolio optimizer to maximize return for any given tolerance of risk.
  - Includes automatically-updated visual aids upon changes in assumptions or risk tolerance.
Option Pricing Model (BlackScholes)
  - Macro-enabled as this is a hockey stick template for a variety of options dependent upon the investor's strategy.
Real Estate Private Equity Model
  - Showcases the returns for an investor into residential real estate, with a multitude of return measures.
  - Automatically updates with changes in assumptions from the parameters tab.
